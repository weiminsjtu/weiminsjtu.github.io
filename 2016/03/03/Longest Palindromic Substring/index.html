Longest Palindromic Substring最长回文子串

  • algorithms
  • leetcode

今年注定会是忙碌的一年，找实习、找工作、发论文、毕业设计……虽然去年就想到要刷leetcode，但是一直都没有着手开始。前两天在知乎看到有人说，“刷了leetcode不见得拿到offer，但不刷leetcode肯定拿不到offer”。呵呵哒，进入狂刷leetcode的状态ing。

Question

Given a string S , find the longest palindromic substring in S . You may assume that the maximum length of S is 1000, and there exists one unique longest palindromic substring.

Manacher算法O(N)

1.首先我们把字符串S改造一下变成T，改造方法是：在S的每个字符之间和S首尾都插入一个”#”。

例如，S=”abaaba”，那么T=”#a#b#a#a#b#a#”。

为什么插入”#”呢？你会发现，长度为奇数和偶数的回文经过处理都变成了奇数。奇数长度便于中心点的处理。

2.按照回文对称中心，以Ti为中心左右扩展才能确定以Ti为中心的回文长度d到底是多少。我们把各个Ti处的回文半径存储到数组P，用P[i]表示以Ti为中心的回文长度。那么当我们求出所有的P[i]，取其中最大值就能找到最长回文子串了。

T = # a # b # a # a # b # a #
P = 0 1 0 3 0 1 6 1 0 3 0 1 0

3.怎么计算P[i]呢？
我们来看一个重叠得更典型的例子，即S=”babcbabcbaccba”。

假设目前for循环遍历，我们已经计算出了一部分P。竖实线表示回文”abcbabcba”的中心C，两个虚实线表示其左右边界L和R。你下一步要计算P[i]，i围绕C的对称点是i’。你有办法高效地计算P[i]吗？

我们先看一下i围绕C的对称点i’（此时i’=9）。

据上图所示，很明显P[i]=P[i’]=1。这是因为i和i’围绕C对称。同理，P[12]=P[10]=0，P[14]=P[8]=0。现在再看i=15处。

此时P[15]=P[7]=7？错了，你逐个字符检测一下会发现此时P[15]应该是5。为什么呢？

如上图所示，两条绿色实线划定的范围必定是对称的，两条绿色虚线划定的范围必定也是对称的。此时请注意P[i’]=7，超过了左边界L。超出的部分就不对称了。此时我们只知道P[i]>=5，至于P[i]还能否扩展，只有通过逐个字符检测才能判定了。(即超出右边界的部分需要逐个字符检测)
在此例中，P[21]≠P[9]，所以P[i]=P[15]=5。

if P[ i' ] < R – i,（未越界）

    P[ i ] ← P[ i' ]

else P[ i ] ≥ R - i
    此时要穿过R逐个字符判定P[i]).

4.C的位置是需要移动的，如果i处的回文超过了R，那么就C=i，同时相应改变L和R即可。

所以，每次求P[i]，都有两种可能。如果P[i‘] < R – i，我们就P[i] = P[i’]。否则，就从R开始逐个字符求P[i]，并更新C及其R。

过程描述如下：
＊P[0],P[1]均为0
＊i＝2，比较T[i+1]和T[i-1]，比较边界长度为i，直到找到不相等情况，此时P[2]=1,令c＝2即中心点,R=c+P[2]=3 L=c-P[2]=1；
＊i=3,P[i’]=P[i-2(i-c)]=P[2c-i]=P[1]=0,R-i=0,所以需要逐个检测字符（原理同上）。最后得到P[i]
*i=4,明显需要逐个检测，得到P[4]=3,令c=4,R=7,L=1.继续……

C的位置始终指的是当前最长回文子串的中心位置。

js code

var longestPalindrome = function(s) {
var t = s.split("").join("#");
t="#" + t + "#";
var P = [0,1],c=1, R=c+P[c], L=c-P[c];

for(var i=1;i<t.length;i++){
    if(P[2*c-i] < R-i){
        P[i] = P[2*c-i];
    }else{
        var count=0;
        for(var j=1;j<=Math.min(i,t.length-i-1);j++){
           if(t[i+j]===t[i-j]){
              count++; 
           }else{
               break;
           }
       }
       P[i]=count;
       if(i+count > R){
           c=i;
           R=c+P[c];
           L=c-P[c];
       }
    }
}
var max = Math.max.apply(null, P);
c = P.indexOf(max);
return t.slice(c-P[c],c+P[c]+1).split('#').join('');
};
