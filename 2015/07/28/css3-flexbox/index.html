CSS3伸缩布局盒模型

  • CSS3
  • Flexbox
  • 盒模型

昨天睡觉前翻了一下大漠的《图解CSS3》，看到CSS3伸缩布局盒模型，好奇心吸引了我，但书上的语言多少有些冗长繁琐，现对这种新的布局模式进行整理学习。

CSS中的布局模式

CSS中的布局模式是由一个盒与其兄弟、祖先盒的关系决定其尺寸与位置的。有以下几种：

1.块布局

2.行内布局

3.表格布局

4.定位布局

CSS3引入了Flexbox布局，提供了一个更加有效的方式制定、调整和分布容器里的项目布局，即使这些项目大小未知或动态变化。

Flex布局？

Flex即Flexible Box，意即“弹性布局”，主要思想是让容器有能力让其子项目改变宽度、高度、顺序从而以最佳方式填充可用空间。比如伸缩项目来扩展填满可用空间，又或者缩小它们以防止溢出容器。

通过display属性声明为flex布局。

display: flex;

display: inline-flex;

//webkit浏览器需添加－webkit前缀
display: -webkit-flex; /* Safari */
display: -webkit-inline-flex; /* Safari */

基本概念

采用Flex布局的元素，成为Flex Container，其子元素为Flex items。

默认情况下，容器存在两根轴：水平的main axis和垂直的cross axis。两者的开始位置和结束为止分别为：main start+main end, cross start+cross end.项目默认按照main axis排列。

* flex-direction
* flex-wrap
* flex-flow
* justify-content
* align-items
* align-content

决定main axis的方向，即项目排列的方向。

Flex Items Properties

* order
* flex-grow
* flex-shrink
* flex-basis
* flex
* align-self

1.order

定义项目的排列顺序，数字越小，越靠前，默认为0.

order: <integer>;

order:

2.flex-grow

定义项目的放大比例，默认为0 (不放大)。

flex-grow: <number>; /* default 0 */

flex-grow:

3.flex-shrink

定义项目的缩小比例，默认为1 (如果空间不足，将缩小)。为0表示不缩小。

flex-shrink: <number>; /* default 1 */

flex-shrink:

4.flex-basis

定义项目在分配剩余空间之前，项目占据的main size。

flex-basis: <length> | auto; /* default auto */

flex-basis:

5.flex

是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。

flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

6.align-self

允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

align-self: auto | flex-start | flex-end | center | baseline | stretch;
